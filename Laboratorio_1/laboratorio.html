Laboratorio

Enunciado

El laboratorio consiste en:

  1. Buscar un conjunto de datos (dataset) en Kaggle
  2. Analizar los datos del problema
  3. Plantear un problema con los datos
  4. Resolver el problema
  5. Crear una presentación web con el problema

A continuación, cada sección.

1

Buscar los datos

El primer paso es buscar un conjunto de datos (dataset) en Kaggle. Para esto, se recomienda utilizar la página de datasets de Kaggle, y buscar un conjunto de datos que sea de su interés.

Para este laboratorio, se recomienda buscar un conjunto de datos que tenga al menos 1000 datos, y que tenga al menos tres columnas de datos que pueden ser tanto categóricos (al menos una) como numéricos.

2

Analizar los datos

El segundo paso es importar y analizar los datos del problema con Python. Para esto, Pandas será la herramienta de manipulación y Matplotlib de graficación.

  1. Importar los datos como un DataFrame
  2. Mostrar las primeras filas del DataFrame
  3. Graficar los datos según sea conveniente (ver tipos de gráficas según tipos de datos). Al menos tres gráficas para mostrar cómo se comportan los datos.

3

Plantear el problema

El tercer paso es plantear un problema con los datos. Para esto, es necesario utilizar los conceptos de probabilidad vistos en el curso, y plantear un problema que pueda ser resuelto con los datos.

Los problemas típicos consisten en encontrar una probabilidad de encontrar un "evento" particular, conociendo las probabilidades necesarias de otros eventos.

4

Resolver el problema

El cuarto paso es resolver el problema computacionalmente. Para esto, es necesario utilizar los conceptos de probabilidad vistos en el curso, y plantear una solución general con un script de Python.

Para resolver el problema, es necesario utilizar Python y las herramientas vistas en el curso.

5

Crear la presentación

El quinto paso es crear una presentación web con el problema. Para esto, es necesario utilizar los conceptos de HTML vistos en el curso, y plantear una presentación web con Reveal.js.

Para crear la presentación, es necesario utilizar HTML y Reveal.js.

Entrega

Presentación

La presentación debe ser una página web con una presentación en Reveal.js (como esta), con:

  • El problema a resolver
  • La solución del problema
  • La explicación de la solución
  • La conclusión del problema
